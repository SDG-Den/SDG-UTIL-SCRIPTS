Package Name: sdg-util
Descriptive Name: SDG Utility Scripts
Source: https://github.com/SDG-Den/SDG-UTIL-SCRIPTS
Maintainer: SDGDen
Version:0.2.2

Dependencies: 
fzf, bat, firefox, tldr (tealdeer), eza, micro

Description: 
Comprehensive suite of terminal-based help tools, documentation browsers, git project launcher, Material You color previewer, RGB keyboard bar, and detailed reference sheets. Commands: tldrtui (command TLDR/man viewer), documentation (online docs + local refs), reference (config reference sheets), colortui (Material You palette preview), git-projects (git project launcher), rgbkeys (RGB keyboard bar). Ships 25 end-user documentation files across 5 doc sets.
