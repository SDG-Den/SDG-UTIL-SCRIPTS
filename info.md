Package Name: sdg-util
Descriptive Name: SDG Utility Scripts
Source: https://git.sdgcloud.nl/SDGDen/SDG-UTIL-SCRIPTS
Maintainer: SDGDen <sdgden@sdgcloud.nl>
Version:0.2

Dependencies: 
fzf, bat, firefox, tldr (tealdeer), eza, micro

Description: 
Comprehensive suite of terminal-based help tools, documentation browsers, git project launcher, Material You color previewer, and detailed reference sheets. Commands: sdg-tldr (command TLDR/man viewer), helptui (topic-based help browser), documentation (online docs + local refs), reference (config reference sheets), colortui (Material You palette preview), git-projects (git project launcher). Ships 25 end-user documentation files across 5 doc sets.
